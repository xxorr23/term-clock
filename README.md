# term-clock
ASCII art Terminal Clock

usage: term-clock option
                -C :    show clock at centre
                -X :    custom x-ordinate
                -Y :    custom y-ordinate
color options:
                -r :    red(default)
                -g :    green
                -y :    yellow
                -b :    blue
                -m :    magenta
                -c :    cyan
                -w :    white
