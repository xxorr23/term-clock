# term-clock
An ASCII art based Terminal Clock




$ term-clock -h

usage: term-clock option

		-C :    show clock at centre
		-X :	custom x-ordinate
		-Y :	custom y-ordinate
color options:

                -r :    red(default)
                -g :    green
                -y :    yellow
                -b :    blue
                -m :    magenta
                -c :    cyan
                -w :    white

Installation:

1.  $ git clone https://github.com/xxorr23/term-clock.git
2.  $ cd term-clock/
3.  $ make
4.  $ sudo make install
